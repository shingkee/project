* Private messages
* Members and community social activity and hashtags
* Friends and Followers
* Real-time notifications
* Members directory
* Front-end registration & login
* Custom user roles
* User profiles
* Members group
* Custom form fields
* Mails and Notifications

= Free Features =

* [Friends and Followers](http://supsystic.com/membership-friends-followers/ "Membership and Community friends and followers option") - Each User Profile of Membership by default includes Friends and Followers tab. On this tabs you can check the list of your friends and followers. Also here you can accept or decline new friends.

* [Private Messages](http://supsystic.com/private-messages/ "Communties private messages") - Create private chats and conversations with membership users.

* [Custom user profiles](http://supsystic.com/membership-profile-customization/ "Cutomize user profile") - Membership by Supsystic plugin provides you a huge list of settings for customizing appearance of the user profile.

* [Front-end registration & login](http://supsystic.com/membership-registration-login/ "Front End user registration and login") - Add any type of the custom user field. Make it required or not. Add Google ReCaptcha service to the registration page. 

* [Members directory](http://supsystic.com/members-directory/ "Members directory") - Allow users to find each other, follow, add to friend, send private message

Control the content of user profiles, create users groups and conversations with your friends. Post your minds, share photos and videos, comment your friend’s posts with Membership by Supsystic Plugin.

== Installation ==

1. Download Membership by Supsystic plugin
2. Unarchive (Extract) membership-by-supsystic.zip
3. Copy the folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the membership-by-supsystic plugin folder in the folder
6. Go to admin panel => open item "Plugins" => activate Membership by Supsystic
